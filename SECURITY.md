# Security Policy

## Supported Versions

AuditPilot is currently under active development.

| Version | Supported |
|----------|-----------|
| main | ✅ |
| older branches | ❌ |

---

## Reporting a Security Vulnerability

Please **do not** open a public GitHub Issue for security vulnerabilities.

Instead, email:

```
YOUR_EMAIL_HERE
```

Include:

- Description of the issue
- Steps to reproduce
- Potential impact
- Screenshots or proof of concept (if applicable)

We aim to acknowledge reports within **72 hours**.

---

## Scope

Examples include:

- Authentication bypass
- Remote code execution
- Arbitrary file upload
- Sensitive information disclosure
- Dependency vulnerabilities
- Path traversal
- Injection attacks

---

## Responsible Disclosure

Please allow reasonable time before publicly disclosing vulnerabilities so they can be investigated and resolved.

Thank you for helping keep AuditPilot secure.
